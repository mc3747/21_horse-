字数统计: 421 | 阅读时长 ≈ 1

1，修改主题

  • 寻找自己喜欢的主题：https://hexo.io/themes/
  • 进入配置blog的目录（选用的next主题）：
    git clone https://github.com/iissnan/hexo-theme-next themes/next
  • 打开_config.yml文件，将“theme”改成“next”

2，验证主题

  • 启动本地调试模式：hexo s –debug
  • 访问网站：http://localhost:4000

3，本地图片未显示

  • 方案1
    • 新建_post文件夹平行的文件夹images，里面放图片
    • 设置属性：post_asset_folder设置成 false不变
    • 相对路径引用图片
    • 结果时有时无
  • 方案2
    • 安装插件：cd /d 你的hexo根目录
      npm install https://github.com/7ym0n/hexo-asset-image –save
    • 设置属性：post_asset_folder设置成 true
    • 运行hexo n “你的博文名字”生成md博文时，/source/_posts目录下会生成一个与md同名的文件夹，将图片放人该文件夹
    • 使用! 插入图片即可
    • 结果时有时无
  • 方案3
    • 使用图床：https://upload.cc

4，添加个人头像

  • 进入文件夹路径themes/next/source/images
  • 添加一张图片avatar.jpg
  • 进入路径themes/next/_config.yml,设置avatar属性:
    • avatar: /images/avatar.jpg

5，添加分类

  • 1，进入blog主文件夹目录，输入 $ hexo new page categories
  • 2, 生成categories文件夹，里面的index.md,添加type属性如下
    • title: 文章分类 date: 2019-10-09 17:11:48 type: "categories"
  • 3，给文章添加“categories”属性，添加categories属性
    • title: hexo搭建个人博客1 date: 2019-09-30 14:26:12 categories: — 工程相关

6，添加标签

  • 与添加分类基本一致
  • 新建命令为：hexo new page tags
  • index.md中新增type： type: "tags"
  • 文章中新增：tags:- jQuery
  • 不同点：一篇文章可以拥有多个tag，但是分类只能嵌套
