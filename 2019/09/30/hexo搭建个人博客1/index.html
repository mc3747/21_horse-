hexo搭建个人博客1

发表于 2019-09-30

hexo配置的参考文档：https://hexo.io/zh-cn/docs/

1，检查安装环境

  • 检查node.js版本
    node -v

  • 检查git版本
    git --version

  • 如果未安装请参考 [git]: https://sourceforge.net/projects/git-osx-installer/

2，安装hexo

npm install -g hexo-cli

3，初始化建立博客框架

  • 在你的家目录下创建一个blog文件夹
    mkdir blog

  • 进入目录
    cd blog

  • 初始化目录
    hexo init

  • 开启本地服务
    hexo s

  • 本地浏览器验证：http://localhost:4000/

4，关联github仓库

  • 新建仓库

    • ⚠️：仓库名一定要与用户名一致，后跟.github.io

  • 配置_config.yml文件

  • 新建文件:hexo new "hexo搭建个人博客-1"

6，生成页面，提交代码

  • 生成静态页面:hexo g

  • 假如报错，提示无法链接到gi或找不到git，运行：
    npm install hexo-deployer-git --save
