1，leanCloud统计阅读次数

  1. 参考文章：https://www.jianshu.com/p/f2ec7eea543d?utm_source=oschina-app
  • 注册leancound账号，下载国际版，可以免费
  • 新建应用，获取应用的APP ID和APP Key
  • 新建class类，类名Counter
  • 修改配置文件next/_config.yml的 leancloud_visitors 属性

2，添加文章头部-字数统计和阅读时长

  1. 参考文章：https://lewky.cn/posts/22195.html
  2. 安装插件：主目录下 npm i –save hexo-wordcount
  3. 修改主题配置文件 _config.yml的 post_wordcount 属性
    • # 文章字数展示设置
      post_wordcount:
      # 文本显示
      item_text: true
      # 文章字数统计
      wordcount: true
      # 阅读时长
      min2read: true
      # 站点总字数统计
      totalcount: false
      # 该post_wordcount的所有设置另起一行显示
      separated_meta: true

4，添加评论

  1. 参加文章：https://blog.csdn.net/qq_28963397/article/details/82910554
  2. 还是使用leanClound项目的appid和appkey
  3. 找到配置文件next/_config.yml的 valine 属性
    • valine:
      enable: true
      appid: xxxxxx # 填写 leancloud appid
      appkey: xxxxxx # 填写 leancloud  appkey
      notify: false #
      verify: false # Verification code
      placeholder: 各位道友，请举手发言 # comment box placeholder

5，添加RSS

  1. 安装插件：npm install hexo-generator-feed –save
  2. 在根目录下的/theme/next/_config.yml文件中添加配置 feed 属性
    • feed: type: rss2
      path: rss2.xml
      limit: 20
      hub:
      content: 'true'
      type: RSS的类型(atom/rss2) path: 文件路径,默认是atom.xml/rss2.xml limit: 展示文章的数量,使用0或则false代表展示全部 content: 在RSS文件中是否包含内容 ,有3个值 true/false默认不填为false
  3. 站点配置/theme/next/_config.yml文件中配置 url 属性
  4. ⚠️没有域名，订阅的rss能看到目录，但访问不了详情

8，添加个人联系方式

  1. 主题配置文件_config.yml中的的social属性打开
  2. 配置如：GitHub: https://github.com/yourname || github，
    • GitHub为展示
    • https://github.com/yourname为跳转链接
    • github为图标
  3. 图标配置网站：https://fontawesome.com/icons?d=gallery
