##1， less

1.1 中文官网

https://less.bootcss.com

1.2 在 Node.js 环境中使用 Less

1. 新建工程：npm init
2. 安装less：npm install -g less
3. 新建less文件：touch styles1.less
4. 编译命令：
    终端显示对应的css命令：lessc styles.less
    输出css文件的命令：lessc styles1.less styles1.css

1.3 在浏览器中使用less

<link rel="stylesheet/less" type="text/css" href="styles.less" />
<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/3.10.0-beta/less.min.js" ></script>

2，Sass

2.1 Sass中文网

https://www.sass.hk

2.2 Sass安装

1. 安装ruby环境（）
    mac自带ruby环境
    查看版本：ruby -v
2. 安装sass    
    //安装如下(如mac安装遇到权限问题需加 sudo gem install sass)
    gem install sass
    gem install compass
3. 查看sass版本：sass -v
4. 查看compass版本：compass -v

2.4 文件生成

1. scss文件直接生成css文件：sass style.scss:style.css
2. 添加编译格式的命令：
    sass --watch input.scss:output.css --sourcemap
