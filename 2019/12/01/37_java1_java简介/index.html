1，发展历史

1. 由SUN公司的James Gosling等人发明，最初命名Oak（橡树）
2. SUN全称(Stanford University Network)
3. SUN在2009年被Oracle以74亿美金收购

2，java分类

JavaSE（J2SE）（Java2 Platform Standard Edition，java平台标准版）
JavaME(J2ME)  (Java 2 Platform Micro Edition，java平台微型版)。
JavaEE(J2EE)  (Java 2 Platform,Enterprise Edition，java平台企业版)

3，版本情况

java版本 = java SE 版本 = JDK的版本
最多的是java 8

4，跨平台原理

4.1 C/C++跨平台原理

生成相应平台的可执行文件

4.1 Java跨平台原理

生成跟平台无关的字节码文件

5，java优势

1. 跨平台（不用担心编译器）
2. 垃圾回收机制（不用手动管理内存，排除内存泄露和循环引用）
3. 生态圈繁荣，
