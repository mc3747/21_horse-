5，盒子模型


1,IE盒模型的content部分包含了border和padding
2,box-sizing属性控制（box-sizing: content-box（让元素维持W3C的标准盒模型） | border-box（此值让元素维持IE传统的Box Model（IE6以下版本）） | inherit）
3,组成部分
    outline
    margin
    border
    padding
    content

  ♨️  轮廓（outline）属性的位置让它不像边框那样参与到文档流中，因此轮廓出现或消失时不会影响文档流，即不会导致文档的重新显示

6，响应式设计
响应式 Web 设计的网页能在所有设备上有好显示。
响应式 Web 设计只使用 HTML 和 CSS。
响应式 Web 设计不是一个程序或Javascript脚本。
