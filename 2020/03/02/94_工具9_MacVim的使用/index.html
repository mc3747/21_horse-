2,下载的MacVim

  1. 下载:https://link.zhihu.com/?target=https%3A//macvim-dev.github.io/macvim/
  2. 安装后验证: 
    :version
  3. 模式切换
    1. 从命令模式->编辑模式：i，a，o，I，A，O
    2. 从编辑模式->命令模式：ESC
    3. 从命令模式->末行模式：输入一个冒号，即shift+；
  4. 命令行模式
    1. :wq 保存后退出vi，
    2. :wq! 则为强制储存后退出（常用）
    3. :w+文件名称 保存但不退出（常用）,相当于新建
    4. :q 离开 vi （常用）
