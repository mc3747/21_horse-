1,参考网站

https://reactnative.cn

2,环境搭建

1,Node
    1. 查看node版本:node -v
    2. 安装node:brew install node
    3. 更新node:brew upgrade node
    4. 保证node在指定的版本(一般要最新版)

2,Watchman
    1_查看watchman:watchman -v 
    2_安装watchman:brew install watchman

3,yarn
    1_yarn是Facebook 提供的替代 npm 的工具，可以加速 node 模块的下载
    2_安装yarn:npm install -g yarn
    3_查看yarn版本:yarn -v

4,Xcode和CocoaPods

5,使用npx代替npm镜像
        # 使用nrm工具切换淘宝源
        npx nrm use taobao

        # 如果之后需要切换回官方源可使用
        npx nrm use npm

6,CocoaPods的镜像

5,常见故障

1_更新pod时候报错:configure: error: C compiler cannot create executables;See `config.log' for more details
  解决:输入:sudo xcode-select --switch /Applications/Xcode.app
2_真机运行时报错:Can't find the 'node' binary to build the React Native bundle.
解决:终端:which node找打node的路径,赋值到以下目录

 3_不识别命令:react-native
    1_安装sudo npm install -g react-native-cli
    2_找到rn的安装路径:which react-native  
    3_全局配置:
        export RN=/usr/local/bin/react-native
        export PATH=$RN:$PATH

4_模拟器没有问题,真机调试报错
    进入iOS项目,在APPDelegate里sourceURLForBridge方法中设置,如下图

5_怎样安装第三方插件
    npm install react-navigation --save

6_rn安装报错
    报错类型:npm ERR! cb() never called! 
        npm ERR! This is an error with npm itself. Please report this error at: 
        npm ERR! <https://npm.community>
    解决方式:
        npm config rm proxy
        npm config rm https-proxy
7_安装了npm install --save react-navigation后报错"Unable to resolve module `react-native-gesture-handler` from `node_modules/@"
    解决方式:安装触摸方式(需要执行以下方法,多增加3个库)
    npm install --save react-native-gesture-handler react-native-reanimated react-native-screens
    react-native link react-native-reanimated
    react-native link react-native-gesture-handler
8_安装完仍然报错:Unable to resolve module ‘react-native-swiper’ from ‘Users/Demo/app/home/Home.js’:Module ‘react-native-swiper’...
根据提示:删除node_modules文件夹,重新安装 yarn install
9_报错:Invariant Violation: Native module cannot be null
解决:需要更新原生的pod
10_使用导航报错,需要额外安装库
    解决: npm install react-native-safe-area-context -save
         npm install @react-native-community/masked-view -save
