123_web24_js的JSON对象

发表于 2020-05-30 | 分类于 前端 | | 阅读次数:
字数统计: 805 | 阅读时长 ≈ 3

1_定义

1.JSON全称JavaScript 对象表示法（JavaScript Object Notation）。是一种严格的JS对象的格式
2.JSON 是存储和交换文本信息的语法。类似 XML。
3.JSON 比 XML 更小、更快，更易解析。

2_特点

1.当作为对象时，用于获取JSON中的数据，
2.而作为字符串时常用于网络数据传输

3_JSON语法规则：

1_数据在名称/值对中
2_数据由逗号分隔
3_花括号保存对象
4_方括号保存数组

4_JSON对象和js对象

1_JSON对象 必须加双引号，而 js 对象属性名可以不加双引号。
2_JSON对象 没有变量声明，而 js 对象有。
3_JSON对象 没有分号，而 js 对象有。

4_JSON对象和js对象

内容放在方括号内

5_序列化操作

1_JSON.stringify()
2_将JavaScript对象转换成JSON字符串
  3_不论是普通的js对象,还是严格的json对象,都会转成标准的json字符串
4_序列化规则
    1.默认情况下，JSON.stringify()输出的JSON字符串不包含任何空格字符或缩进，因此结果就像上面那样。
    2.⚠️序列化JavaScript对象时，所有函数及原型成员都会被忽略，不体现在结果上。
    3.值为undefined的任何属性都会被跳过
5_序列化过滤器
    1_若过滤器的第二个参数是数组，则JSON.stringify()返回的结果将只包含数组中的属性
        var leo =  {
            name : "leo",
            age  : 25,
            box  : ["a","b","c"]
        }
        var pingan = JSON.stringify(leo,["name","age"]);
        console.log(pingan); // "{"name":"leo","age":25}"
    2_过滤器的第二个参数是函数，则情况就不一样了，传入的函数需有两个参数（属性名和属性值
        var leo =  {
        "name" : "leo",
        "age"  : 25,
        "box"  : ["a","b","c"]
    }
    var pingan = JSON.stringify(leo,function(key, value){
        switch(key){
            case "name":
                return "我叫" + value
            case "age":
                return value + "岁"
            default:
                return value
        }
    });
    console.log(pingan); // "{"name":"我叫leo","age":"25岁","box":["a","b","c"]}"

6_选项JSON.stringify()第三个参数是个选项，控制结果中的缩进和空白符
    若选项只有一个值，则表示每个级别缩进的空格数
    var leo =  {
        "name" : "leo",
        "age"  : 25,
        "box"  : ["a","b","c"]
    }
    var pingan = JSON.stringify(leo, null, 4);
    console.log(pingan);
    /*
    "{
        "name": "leo",
        "age": 25,
        "box": [
            "a",
            "b",
            "c"
        ]
    }"
    */

6_反序列化操作

1_JSON.parse()
2_如果字符串不是有效的JSON，则会抛出错误
3_示例
    let pingan = '{"name":"leo","age":25,"box":["a","b","c"]}'
    let copyPingan = JSON.parse(pingan);
    console.log(copyPingan); // { name: 'leo', age: 25, box: [ 'a', 'b', 'c' ] }
4_解析函数(还原函数)
    1_还原函数接收两个参数：属性名和属性值
    2_示例
        var leo =  {
            "name" : "leo",
            "age"  : 25,
            "date" : new Date(1993, 9, 9)
        }
        var pingan = JSON.stringify(leo);
        var copy = JSON.parse(pingan,function (key, value){
            // return key == "date" ? new Date(value) : value;
            if(key == "date"){
                return new Date(value);
            }else{
                return value; 
            }
        })
        console.log(copy);
