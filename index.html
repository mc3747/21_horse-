hexo搭建个人博客2

发表于 2019-09-30

1，修改主题

  • 寻找自己喜欢的主题：https://hexo.io/themes/
  • 进入配置blog的目录（选用的next主题）：
    git clone https://github.com/iissnan/hexo-theme-next themes/next
  • 打开_config.yml文件，将“theme”改成“next”

2，验证主题

  • 启动本地调试模式：hexo s –debug
  • 访问网站：http://localhost:4000

3，本地图片未显示

  • 方案1
    • 新建_post文件夹平行的文件夹images，里面放图片
    • 设置属性：post_asset_folder设置成 false不变
    • 相对路径引用图片
    • 结果时有时无
  • 方案2
    • 安装插件：cd /d 你的hexo根目录
      npm install https://github.com/7ym0n/hexo-asset-image –save
    • 设置属性：post_asset_folder设置成 true
    • 运行hexo n “你的博文名字”生成md博文时，/source/_posts目录下会生成一个与md同名的文件夹，将图片放人该文件夹
    • 使用! 插入图片即可
    • 结果时有时无
  • 方案3
    • 使用图床：https://upload.cc

4，添加个人头像

  • 进入文件夹路径themes/next/source/images
  • 添加一张图片avatar.jpg
  • 进入路径themes/next/_config.yml,设置avatar属性:
    • avatar: /images/avatar.jpg

发表于 2019-09-30

hexo配置的参考文档：https://hexo.io/zh-cn/docs/

1，检查安装环境

  • 检查node.js版本
    node -v

  • 检查git版本
    git --version

  • 如果未安装请参考 [git]: https://sourceforge.net/projects/git-osx-installer/

2，安装hexo

npm install -g hexo-cli

3，初始化建立博客框架

  • 在你的家目录下创建一个blog文件夹
    mkdir blog

  • 进入目录
    cd blog

  • 初始化目录
    hexo init

  • 开启本地服务
    hexo s

  • 本地浏览器验证：http://localhost:4000/

4，关联github仓库
