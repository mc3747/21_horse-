1,元素的定位

  1,设置属性position : static | relative | absolute | fixed

    2,静态定位:static : 元素框正常生成

    3,相对定位:relative : 
      1_元素相对自己的位置
      2_可以使用top,bottom,left,right四个属性
      3_没有脱离文档流  

    4,绝对定位:absolute : 元素框从文档流完全删除，并相对于其包含块定。包含块可能是文档中的另一个元素或初始包含块。
      1_相对于其最接近的一个具有定位属性的父包含块进行绝对定位
    2_设置top,bottom,left,right四个属性
    3_脱离了文档流  

  5,固定定位:fixed : 类似 absolute，不过其包含块是视窗本身。

2，浮动：
    float（left，right，none，inherit）
    clear（left，right，both，none，inherit），是否清除浮动元素

3，布局
    1，flow布局（流动布局）：网页默认布局
        块级元素都会在所处的最近父级容器元素内自上而下按顺序垂直顺延分布
        内联元素都会在所处的最近父级容器元素内从左到右水平分布显示
    2，    float布局（浮动布局）：让该元素摆脱当前文档流，成为浮动元素
    3，    Layer布局
        绝对定位(position: absolute)



        相对定位(position: relative)
            相对定位与绝对定位最大的区别在于，前者没有脱离当前文档流而后者已经脱离了当前文档流
            可以使用top,bottom,left,right四个属性
        固定定位(position: fixed)
            它的相对移动的坐标是视图（屏幕内的网页窗口）本身。由于视图本身是固定的，它不会随浏览器窗口的滚动条滚动而变化，除非你在屏幕中移动浏览器窗口的屏幕位置，或改变浏览器窗口的显示大小

        流式布局

1.元素的分类

块级:block-level
    1.有<div>,<p>,<li>,<table>
        块级并不都是display属性为block
        div的display属性为block
        li的display属性为list-item
        table的display属性为table
 内联:inline 
    1.有<a>,<span>
        display为inline

2.width和height的作用

width:
    content,padding,border,margin
    box-sizing: content-box;
    min-width和max-width
    width:auto的使用
    怎样宽度分离,父子宽度分离
height:
    height:auto,在绝对定位中
    如何让元素支持 height:100%效果
        1.父元素设置显示的高度值,height: 100%
        2.使用绝对定位
             div {
                  height: 100%;
                  position: absolute;
            }
            绝对定位的宽高百分比计算是相对于 padding box 的
            非绝对定位元素则是相对于 content box 计算的

3,float

4.清楚浮动

clear:right

3.溢出相关

1_overflow:(也可以单独设置overflow-x,overflow-y)
    visible    默认值。内容不会被修剪，会呈现在元素框之外。
    hidden    
        标准流:内容会被修剪，并且其余内容是不可见的。
        浮动后:父容器的overflow设置为hidden,父容器自动包裹子元素
    auto:超出有滚动条,不超出没有滚动条
    scroll    内容会被修剪，一直显示滚动条

2_word-break(单词内怎样断句)
    normal    使用浏览器默认的换行规则。
    break-word 保证单词的完整性
    break-all    允许在单词内换行。
    keep-all    只能在半角空格或连字符处换行。

3_word-wrap(展示是否单词内断句)
    normal    只在允许的断字点换行（浏览器保持默认处理）。
    break-word    在长单词或 URL 地址内部进行换行。

4_white-space
    normal    默认。空白会被浏览器忽略。
    pre    空白会被浏览器保留。其行为方式类似 HTML 中的 <pre> 标签。
    nowrap    文本不会换行，文本会在在同一行上继续，直到遇到 <br> 标签为止。

5_text-overflow
    clip    修剪文本。
    ellipsis    显示省略符号来代表被修剪的文本。
6_配合使用
 white-space: nowrap; overflow: hidden; /*text-overflow: clip;*/ text-overflow: ellipsis;

    function judgeDataType(a){
            let type = Object.prototype.toString.call(a);
            if(type === '[object Null]'){
                console.log('null_空对象')

            }else if(type === '[object Undefined]'){
                console.log('Undefined_未定义')

            }else if(type === '[object Number]'){
                console.log('Number_数字')

            }else if(type === '[object Boolean]'){
                console.log('Boolean_布尔')

            }else if(type === '[object String]'){
                    console.log('String_字符串')    
            }else if(type === '[object Array]'){
                console.log('Array_数组')

            }else if(type === '[object Object]'){
                console.log('Object_对象')

            }else if(type === '[object Function]'){
                console.log('Function_函数')
            }
    }
