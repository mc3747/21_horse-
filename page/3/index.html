86_方法论5_和领导沟通的艺术

发表于 2020-01-11 | 分类于 其他 | | 阅读次数:
字数统计: 390 | 阅读时长 ≈ 1

1，为什么要学会跟领导沟通

1，老板都喜欢用得心应手的人，利于职场晋升
2，沟通会明确自己工作的方向
3，让自己学习地更多

2，怎样心理上克服恐惧

2.1，恐慌的原因

1，领导自带距离感
2，员工能力不够
3，和领导年龄有代沟

2.2，恐慌的克服方法：

1，逐步脱敏，从小事处沟通开始；
2，去掉领导的社会属性，平等沟通；
3，提前准备，才能及时汇报
4，心理上，要认为跟高层次人沟通会提升自己

3，怎样学会向领导汇报

1，沟通如果是要钱要人，让老板看成绩；只是刷存在感，重点说思路；
2，领导交待的事，不论是否顺利都要及时汇报；
    接手交待的事情，要主动问老板3个问题：
    1，工作目标是什么，或者说怎样做才算完成，
    2，截止时间是什么
    3，有什么资源提供
3，给领导汇报方案，要提供2个以上的方案
    包含优势和劣势，是否多快好省
    多角度，可以体现自己的思考
    给老板做选择题，沟通会更顺畅
4，突出重点，逻辑清晰
5，先讲结论，再展开说明
    有ppt，数据，或者表格之类支撑会更好
6，讲究说话方式
    不能当好好先生，什么都点头，马屁精；
    也不能过于直接，不走大脑；

1，长度

绝对长度
    in，英寸（1in = 96px = 2.54cm）
    cm，厘米
    mm，毫米
    pt，points（1pt = 1/72英寸，也叫磅）
    pc，Picas（1pt = 1/6in）

相对长度
    px，Pixels像素
    em，元素的字体高度
    ex，x-height，字母x的高度
    %，百分比，均基于父元素的
    ch，字符0的宽度
    rem，根元素（html）的font-size
    vw，viewpoint width，视窗宽度，1vw=视窗宽度的1%
    vh，viewpoint height，视窗高度，1vh=视窗高度的1%        vmin，vw和vh中较小的那个
    vmax，vw和vh中较大的那个

em注意点
    基于当前元素的(如果没设置就是继承其父元素的)font-size
    em默认值，即1em = 16px
    em单位具有级联关系
    例子：span标签的结果就是font-size: 24px
    <div style="font-size: 12px;">
        <span style="font-size: 2em">em单位</span>
    </div>

2，字体

在线字体工具：https://www.w3cschool.cn/tools/index?name=csscreate
字体（font-family）：css5种通用字体
Serif 字体（末端拥有额外的装饰）
Sans-serif 字体（末端没有额外的装饰）
Monospace 字体（所有的等宽字符具有相同的宽度）
Cursive 字体
Fantasy 字体
加粗（font-weight）：normal（400），bold（700），900等
大小（ font-size）
文字样式（font-style）：normal ，italic（斜体），oblique
字体后备机制：
例如p{font-family:”Times New Roman”, Times, serif}
font-family属性是多种字体的名称，作为一个”应变”制度，以确保浏览器/操作系统之间的最大兼容性

3，颜色

1，16进制：#ff0000
2，rgb：
    数字：RGB（0,0,255
    百分比：RGB（0％，0％，100％）
3，rgba
    Alpha参数是一个介于0.0（完全透明）和1.0（完全不透明）之间的参数
4，HSL
    HSL（色调，饱和度，明度）
5，HSLA

4,布局

1，布局属性
2，浮动：
    float（left，right，none，inherit）
    clear（left，right，both，none，inherit），是否清除浮动元素
3，定位
    position : static | relative | absolute | fixed
    static : 元素框正常生成
    relative : 元素框偏移某个距离
    absolute : 元素框从文档流完全删除，并相对于其包含块定。包含块可能是文档中的另一个元素或初始包含块。

fixed : 类似 absolute，不过其包含块是视窗本身。
1，flow布局（流动布局）：网页默认布局
块级元素都会在所处的最近父级容器元素内自上而下按顺序垂直顺延分布
内联元素都会在所处的最近父级容器元素内从左到右水平分布显示
2， float布局（浮动布局）：让该元素摆脱当前文档流，成为浮动元素
3， Layer布局
绝对定位(position: absolute)
相对于其最接近的一个具有定位属性的父包含块进行绝对定位
相对定位(position: relative)
相对定位与绝对定位最大的区别在于，前者没有脱离当前文档流而后者已经脱离了当前文档流
固定定位(position: fixed)
它的相对移动的坐标是视图（屏幕内的网页窗口）本身。由于视图本身是固定的，它不会随浏览器窗口的滚动条滚动而变化，除非你在屏幕中移动浏览器窗口的屏幕位置，或改变浏览器窗口的显示大小

提高指定样式规则的应用优先权（优先级）。语法格式{ cssRule !important }，即写在定义的最后面，例如：box{color:red !important;}

    Never 永远不要在全站范围的 css 上使用 !important

3，选择器

css选择器：<https://www.w3cschool.cn/css/css-selector.html>

3.1 选择器分类

1，组合选择器
    分组选择器（相同的样式用逗号，相同）：div,p
    后代选取器(以空格分隔)：div p
    子元素选择器(以大于号分隔）：div>p
    相邻兄弟选择器（以加号分隔）：div+p
    普通兄弟选择器（以波浪号分隔）：div~p
2，属性选择器
    a[title] {color: green;}
3，伪类（pseudo-class）
    由于状态的变化是非静态的，所以元素达到一个特定状态时，它可能得到一个伪类的样式；当状态改变时，它又会失去这个样式。由此可以看出，它的功能和class有些类似，但它是基于文档之外的抽象，所以叫伪类
    a:link {color:#FF0000;} 
    a:visited {color:#00FF00;} 
    a:hover {color:#FF00FF;} 
    a:active {color:#0000FF;}
4，伪元素（pseudo-element）
    CSS伪元素控制的内容和元素是没有差别的，但是它本身只是基于元素的抽象，并不存在于文档中，所以称为伪元素
    p:first-line 
        {
        color:#ff0000;
        font-variant:small-caps;
        }

  ♻️  普通选择器和伪类/伪元素的区别：
普通选择器是直接从HTML文档的DOM树中获取元素的。
而伪类和伪元素选择器是预定义的，且是独立文档元素的,即基于元素的特征，不可以从dom处推断，而是根据交互状态

  ♻️  伪类和伪元素的区别
相同：伪类和伪元素都不出现在源文件和文档树中。也就是说在html源文件中是看不到伪类和伪元素的。
不同：伪类其实就是基于普通DOM元素而产生的不同状态，他是DOM元素的某一特征。伪元素能够创建在DOM树中不存在的抽象对象，而且这些抽象对象是能够访问到的。

3.2 选择器优先级

选择器指向的越准确，它的优先级就越高
通用选择器（*）：
标签选择器：1
伪元素：1
类选择器：10
属性选择器：10
伪类：10
id选择器：100
行内样式：1000

3.3 选择器效率，由高到低

1.id选择器（#myid）
2.类选择器（.myclassname）
3.标签选择器（div,h1,p）
4.相邻选择器（h1+p）
5.子选择器（ul > li）
6.后代选择器（li a）
7.通配符选择器（*）
8.属性选择器（a[rel="external"]）
9.伪类选择器（a:hover, li:nth-child）
    提示选择器的性能

3.4 提升选择器的效率：

关键：所谓高效的CSS就是让浏览器在查找style匹配的元素的时候尽量进行少的查找
1、避免使用通用选择器
2、避免使用标签或 class 选择器限制 id 选择器
    DIV#divBox
3、避免使用标签限制 class 选择器
    p.red{color:red;}
