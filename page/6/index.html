  ⚠️  !important 作用：
提高指定样式规则的应用优先权（优先级）。语法格式{ cssRule !important }，即写在定义的最后面，例如：box{color:red !important;}
  ⚠️  !important 注意：
    Always 要优化考虑使用样式规则的优先级来解决问题而不是 !important
    Only 只在需要覆盖全站或外部 css（例如引用的 ExtJs 或者 YUI ）的特定页面中使用 !important
    Never 永远不要在全站范围的 css 上使用 !important
    Never 永远不要在你的插件中使用 !important

3.1 选择器分类

1，组合选择器
    分组选择器（相同的样式用逗号，相同）：div,p
    后代选取器(以空格分隔)：div p
    子元素选择器(以大于号分隔）：div>p
    相邻兄弟选择器（以加号分隔）：div+p
    普通兄弟选择器（以波浪号分隔）：div~p
2，属性选择器
    a[title] {color: green;}
3，伪类（pseudo-class）
    由于状态的变化是非静态的，所以元素达到一个特定状态时，它可能得到一个伪类的样式；当状态改变时，它又会失去这个样式。由此可以看出，它的功能和class有些类似，但它是基于文档之外的抽象，所以叫伪类
    a:link {color:#FF0000;} 
    a:visited {color:#00FF00;} 
    a:hover {color:#FF00FF;} 
    a:active {color:#0000FF;}
4，伪元素（pseudo-element）
    CSS伪元素控制的内容和元素是没有差别的，但是它本身只是基于元素的抽象，并不存在于文档中，所以称为伪元素
    p:first-line 
        {
        color:#ff0000;
        font-variant:small-caps;
        }

  ♻️  普通选择器和伪类/伪元素的区别：
普通选择器是直接从HTML文档的DOM树中获取元素的。
而伪类和伪元素选择器是预定义的，且是独立文档元素的,即基于元素的特征，不可以从dom处推断，而是根据交互状态

  ♻️  伪类和伪元素的区别
相同：伪类和伪元素都不出现在源文件和文档树中。也就是说在html源文件中是看不到伪类和伪元素的。
不同：伪类其实就是基于普通DOM元素而产生的不同状态，他是DOM元素的某一特征。伪元素能够创建在DOM树中不存在的抽象对象，而且这些抽象对象是能够访问到的。

74_设计2_像素&尺寸&分辨率

发表于 2020-01-05 | 分类于 产品设计 | | 阅读次数:

1 概念：

px：pixel，像素，电子屏幕上组成一幅图画或照片的最基本单元
pt: point，点，印刷行业常用单位，等于1/72英寸
ppi: pixel per inch，每英寸像素数，该值越高，则屏幕越细腻
dpi: dot per inch，每英寸多少点，该值越高，则图片越细腻
dp: dip，Density-independent pixel, 是安卓开发用的长度单位，1dp表示在屏幕像素点密度为160ppi时1px长度
sp: scale-independent pixel，安卓开发用的字体大小单位。

2 根据屏幕分辨率，求ppi

ppi= 屏幕对角线上的像素点数/对角线长度 = √ （屏幕横向像素点^2 + 屏幕纵向像素点^2）/对角线长度

3 查看硬件设备的网站：

https://www.mydevice.io，用终端登陆后，就可以知道该终端的尺寸信息

1，元素的属性

全局属性参考：<https://www.w3cschool.cn/htmltags/ref-standardattributes.html>

1.1 常见的全局属性：

1，id属性
    用于html元素的唯一标识，css使用className
    url使用锚点访问：https://foo.com/index.html#bar，滚动到id为bar的元素的位置
2，class属性
    表示具有相同属性
    不唯一
    可以叠加，中间用空格
3，title属性
    鼠标悬浮，显示的title
4，tabindex
    表示用户按下 Tab 键的时候，网页焦点转移的顺序
5, accesskey
    1，网页元素获得焦点的快捷键
    2，例如：<button accesskey="s">提交</button>
    3，Chrome 浏览器在 Windows 系统和 Linux 系统的快捷键是Alt + 字符键，在 Mac 系统的快捷键是Ctrl + Alt + 字符键
    4，accesskey如果跟操作系统或浏览器级别的快捷键有冲突，这时不会生效
6，hidden
    1，布尔属性，表示当前的网页元素不再跟页面相关，因此浏览器不会渲染这个元素，所以就不会在网页中看到它
    2，<p hidden>本句不会显示在页面上。</p>
    3，CSS 的可见性设置，高于hidden属性。如果 CSS 设为该元素可见，hidden属性将无效
7，lang
    1，网页元素使用的语言
    2，类别
        zh：中文
        zh-Hans：简体中文
        zh-Hant：繁体中文
        en：英语
8，dir
    1，表示文字的阅读方向
    2，可选值
        ltr：从左到右阅读，比如英语。
        rtl：从右到左阅读，阿拉伯语、波斯语、希伯来语都属于这一类。
        auto：浏览器根据内容的解析结果，自行决定
9，contenteditable
    1，允许用户修改内容
    2，可选值
        true或空字符串：内容可以编辑
        false：不可以编辑
10，spellcheck
    1，拼写检查
    2，可选值
11，data-*
    *可以被取代，为自定义属性，用来传参等
12,    style
    元素的 CSS 样式
13,事件处理属性
    见下边第三部分

2，标签的分类

全部标签参考：<https://www.w3cschool.cn/htmltags/html-reference.html>

2.1 文本标签

<br/>：换行
    行内标签换行，用<br/>;
    块级标签换行，使用css样式控制距离；
<wbr/>
    表示一个可选的断行。如果一行的宽度足够，则不断行；如果宽度不够，需要断行;
<pre>
    表示保留原来的格式（preformatted）
<strong>和<b>：粗体
    浏览器会加粗显示。(Boldface)
    不推荐使用<b>，Bold
<em>和<i>：斜体
    表示强调（emphasize），浏览器会以斜体显示它包含的内容
    不推荐使用<i>，Italic 
<sub>，<sup>，<var>：数学相关
    <sub>标签将内容变为下标，bottom
    <sup>标签将内容变为上标，peak
    <var>标签表示代码或数学公式的变量
<u>，<s>：下划线和删除线
    <u>会产生下划线：underline
    <s>标签是一个行内元素，为内容加上删除线：strike

2，百度

1，精准搜索，加上引号“”，
    例子“香蕉牛奶”
2，添加书名号《》，一是书名号会出现在搜索结果中；二是被书名号扩起来的内容，不会被拆分
    例子:《香蕉牛奶》
3，包含特定查询词“+”
    例子:厨子戏子 +厨子
4，排除关键词“-”
    例子:厨子戏子 -痞子
5，百度高级搜索：<http://www.baidu.com/gaoji/advanced.html>
6，把搜索范围限定在网页标题中——intitle:标题
    例子：iOS intitle:零基础
7，把搜索范围限定在特定站点中——site:站名
    例子：五月天 site:www.dytt8.net
8，把搜索范围限定在[url]url链接中——inurl:链接
    例子:inurl:admin
9，Filetype搜索范围限定在指定文档格式中
    例子:中国电信 filetype:doc

3，谷歌：

1，逻辑与：空格表示
    例子：苹果 癌症
2，逻辑或：OR表示
    例子：apple ORmother
3，site：可以查询某个网站页面的收录情况，
    例子：apple site:http://www.apple.com/查看包含关键词apple的网页收录情况
4，link: 查询所有和某个URL有链接的网页，只能单独使用
    例子：link:http://www.apple.com/
5，inurl：
    apple inurl:happy返回结果为包含happy的URL，其它任何地方则包含了apple
6，intitle：
    apple intitle:happy返回结果为包含happy的标题，其它任何地方则包含了apple
7，intext：
    apple intitle:happy返回结果为包含happy的正文，其它任何地方则包含了apple    
8，allinurl:查询返回的网页链接中包含所有查询的关键词
9，allintitle：查询返回的网页标题中包含所有查询的关键词
