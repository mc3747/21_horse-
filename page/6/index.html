1，长度

绝对长度
    in，英寸（1in = 96px = 2.54cm）
    cm，厘米
    mm，毫米
    pt，points（1pt = 1/72英寸，也叫磅）
    pc，Picas（1pt = 1/6in）

相对长度
    px，Pixels像素
    em，元素的字体高度
    ex，x-height，字母x的高度
    %，百分比，均基于父元素的
    ch，字符0的宽度
    rem，根元素（html）的font-size
    vw，viewpoint width，视窗宽度，1vw=视窗宽度的1%
    vh，viewpoint height，视窗高度，1vh=视窗高度的1%        vmin，vw和vh中较小的那个
    vmax，vw和vh中较大的那个

em注意点
    基于当前元素的(如果没设置就是继承其父元素的)font-size
    em默认值，即1em = 16px
    em单位具有级联关系
    例子：span标签的结果就是font-size: 24px
    <div style="font-size: 12px;">
        <span style="font-size: 2em">em单位</span>
    </div>

2，字体

在线字体工具：https://www.w3cschool.cn/tools/index?name=csscreate
字体（font-family）：css5种通用字体
Serif 字体（末端拥有额外的装饰）
Sans-serif 字体（末端没有额外的装饰）
Monospace 字体（所有的等宽字符具有相同的宽度）
Cursive 字体
Fantasy 字体
加粗（font-weight）：normal（400），bold（700），900等
大小（ font-size）
文字样式（font-style）：normal ，italic（斜体），oblique
字体后备机制：
例如p{font-family:”Times New Roman”, Times, serif}
font-family属性是多种字体的名称，作为一个”应变”制度，以确保浏览器/操作系统之间的最大兼容性

3，颜色

1，16进制：#ff0000
2，rgb：
    数字：RGB（0,0,255
    百分比：RGB（0％，0％，100％）
3，rgba
    Alpha参数是一个介于0.0（完全透明）和1.0（完全不透明）之间的参数
4，HSL
    HSL（色调，饱和度，明度）
5，HSLA
6，预定义颜色名称
    147个颜色=17的标准颜色+130个其他颜色，例如Black，Blue等

4,布局

1，布局属性
2，浮动：
    float（left，right，none，inherit）
    clear（left，right，both，none，inherit），是否清除浮动元素
3，定位
    position : static | relative | absolute | fixed
    static : 元素框正常生成
    relative : 元素框偏移某个距离
    absolute : 元素框从文档流完全删除，并相对于其包含块定。包含块可能是文档中的另一个元素或初始包含块。

fixed : 类似 absolute，不过其包含块是视窗本身。
1，flow布局（流动布局）：网页默认布局
块级元素都会在所处的最近父级容器元素内自上而下按顺序垂直顺延分布
内联元素都会在所处的最近父级容器元素内从左到右水平分布显示
2， float布局（浮动布局）：让该元素摆脱当前文档流，成为浮动元素
3， Layer布局
绝对定位(position: absolute)
相对于其最接近的一个具有定位属性的父包含块进行绝对定位
相对定位(position: relative)
相对定位与绝对定位最大的区别在于，前者没有脱离当前文档流而后者已经脱离了当前文档流
固定定位(position: fixed)
它的相对移动的坐标是视图（屏幕内的网页窗口）本身。由于视图本身是固定的，它不会随浏览器窗口的滚动条滚动而变化，除非你在屏幕中移动浏览器窗口的屏幕位置，或改变浏览器窗口的显示大小

  ⚠️  !important 作用：
提高指定样式规则的应用优先权（优先级）。语法格式{ cssRule !important }，即写在定义的最后面，例如：box{color:red !important;}
  ⚠️  !important 注意：
    Always 要优化考虑使用样式规则的优先级来解决问题而不是 !important
    Only 只在需要覆盖全站或外部 css（例如引用的 ExtJs 或者 YUI ）的特定页面中使用 !important
    Never 永远不要在全站范围的 css 上使用 !important
    Never 永远不要在你的插件中使用 !important

css选择器：<https://www.w3cschool.cn/css/css-selector.html>
c3选择器：<https://www.w3cschool.cn/css3/css3-selector.html>

3.1 选择器分类

1，组合选择器
    分组选择器（相同的样式用逗号，相同）：div,p
    后代选取器(以空格分隔)：div p
    子元素选择器(以大于号分隔）：div>p
    相邻兄弟选择器（以加号分隔）：div+p
    普通兄弟选择器（以波浪号分隔）：div~p
2，属性选择器
    a[title] {color: green;}
3，伪类（pseudo-class）
    由于状态的变化是非静态的，所以元素达到一个特定状态时，它可能得到一个伪类的样式；当状态改变时，它又会失去这个样式。由此可以看出，它的功能和class有些类似，但它是基于文档之外的抽象，所以叫伪类
    a:link {color:#FF0000;} 
    a:visited {color:#00FF00;} 
    a:hover {color:#FF00FF;} 
    a:active {color:#0000FF;}
4，伪元素（pseudo-element）
    CSS伪元素控制的内容和元素是没有差别的，但是它本身只是基于元素的抽象，并不存在于文档中，所以称为伪元素
    p:first-line 
        {
        color:#ff0000;
        font-variant:small-caps;
        }

  ♻️  普通选择器和伪类/伪元素的区别：
普通选择器是直接从HTML文档的DOM树中获取元素的。
而伪类和伪元素选择器是预定义的，且是独立文档元素的,即基于元素的特征，不可以从dom处推断，而是根据交互状态

  ♻️  伪类和伪元素的区别
相同：伪类和伪元素都不出现在源文件和文档树中。也就是说在html源文件中是看不到伪类和伪元素的。
不同：伪类其实就是基于普通DOM元素而产生的不同状态，他是DOM元素的某一特征。伪元素能够创建在DOM树中不存在的抽象对象，而且这些抽象对象是能够访问到的。

3.2 选择器优先级

选择器指向的越准确，它的优先级就越高

1.id选择器（#myid）

    提示选择器的性能

关键：所谓高效的CSS就是让浏览器在查找style匹配的元素的时候尽量进行少的查找

74_设计2_像素&尺寸&分辨率

发表于 2020-01-05 | 分类于 产品设计 | | 阅读次数:

1 概念：

px：pixel，像素，电子屏幕上组成一幅图画或照片的最基本单元
pt: point，点，印刷行业常用单位，等于1/72英寸
ppi: pixel per inch，每英寸像素数，该值越高，则屏幕越细腻
dpi: dot per inch，每英寸多少点，该值越高，则图片越细腻
dp: dip，Density-independent pixel, 是安卓开发用的长度单位，1dp表示在屏幕像素点密度为160ppi时1px长度
sp: scale-independent pixel，安卓开发用的字体大小单位。

2 根据屏幕分辨率，求ppi

ppi= 屏幕对角线上的像素点数/对角线长度 = √ （屏幕横向像素点^2 + 屏幕纵向像素点^2）/对角线长度

3 查看硬件设备的网站：

https://www.mydevice.io，用终端登陆后，就可以知道该终端的尺寸信息

1，元素的属性

全局属性参考：<https://www.w3cschool.cn/htmltags/ref-standardattributes.html>

1.1 常见的全局属性：

1，id属性
    用于html元素的唯一标识，css使用className
    url使用锚点访问：https://foo.com/index.html#bar，滚动到id为bar的元素的位置
2，class属性
    表示具有相同属性
    不唯一
    可以叠加，中间用空格
3，title属性
    鼠标悬浮，显示的title
4，tabindex
    表示用户按下 Tab 键的时候，网页焦点转移的顺序
5, accesskey
    1，网页元素获得焦点的快捷键
    2，例如：<button accesskey="s">提交</button>
    3，Chrome 浏览器在 Windows 系统和 Linux 系统的快捷键是Alt + 字符键，在 Mac 系统的快捷键是Ctrl + Alt + 字符键
    4，accesskey如果跟操作系统或浏览器级别的快捷键有冲突，这时不会生效
6，hidden
    1，布尔属性，表示当前的网页元素不再跟页面相关，因此浏览器不会渲染这个元素，所以就不会在网页中看到它
    2，<p hidden>本句不会显示在页面上。</p>
    3，CSS 的可见性设置，高于hidden属性。如果 CSS 设为该元素可见，hidden属性将无效
7，lang
    1，网页元素使用的语言
    2，类别
        zh：中文
        zh-Hans：简体中文
        zh-Hant：繁体中文
        en：英语
8，dir
    1，表示文字的阅读方向
    2，可选值
        ltr：从左到右阅读，比如英语。
        rtl：从右到左阅读，阿拉伯语、波斯语、希伯来语都属于这一类。
        auto：浏览器根据内容的解析结果，自行决定
9，contenteditable
    1，允许用户修改内容
    2，可选值
        true或空字符串：内容可以编辑
        false：不可以编辑
10，spellcheck
    1，拼写检查
    2，可选值
11，data-*
    *可以被取代，为自定义属性，用来传参等
12,    style
    元素的 CSS 样式
13,事件处理属性

2，标签的分类

全部标签参考：<https://www.w3cschool.cn/htmltags/html-reference.html>

2.1 文本标签

<br/>：换行
    行内标签换行，用<br/>;
    块级标签换行，使用css样式控制距离；
<wbr/>
    表示一个可选的断行。如果一行的宽度足够，则不断行；如果宽度不够，需要断行;
<pre>

<em>和<i>：斜体

    <sub>标签将内容变为下标，bottom
    <sup>标签将内容变为上标，peak
    <var>标签表示代码或数学公式的变量
<u>，<s>：下划线和删除线
    <u>会产生下划线：underline
    <s>标签是一个行内元素，为内容加上删除线：strike
